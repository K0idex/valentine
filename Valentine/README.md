# Valentine Mobile-First Website

This is a simple, mobile-first Valentine-themed static site.

How to preview
- Open `index.html` in your browser, or from the project folder run:

```bash
python -m http.server 8000
# then open http://localhost:8000 in your phone or browser
```

Personalization
- Edit `script.js` -> the `paragraphs` array to change the 5–7 messages.
- Add personal images to the `images/` folder and update `invite.html` to include them.
- Change `invite.html` mailto address to her real email or replace with a form.

Files
- `index.html` — landing page with sequential typewriter messages.
- `styles.css` — mobile-first styling and romantic theme.
- `script.js` — typewriter logic and CTA navigation.
- `invite.html` — invitation page and RSVP button.
- `images/` — add personal photos here.
